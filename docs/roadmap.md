# Roadmap

## Milestone 1：单机播放器

目标：即使没有联网，也能作为稳定的本地播放器使用。

交付：

- 桌面应用壳。
- 本地视频播放。
- 基础控制栏。
- 快捷键。
- 字幕加载和偏移。
- 播放列表本地管理。

验收：

- 能播放常见 `mp4`、`mkv`。
- 播放器控制和快捷键行为一致。
- 字幕偏移即时生效。

## Milestone 2：房间和文件匹配

目标：两个人能进入同一房间，并确认正在播放同一份媒体。

交付：

- 信令服务。
- 房间创建和加入。
- 短期断线重连。
- 文件元数据读取。
- 分段指纹匹配。
- 本地路径绑定。

验收：

- 两个客户端能同时进入房间。
- 相同文件能自动匹配。
- 不同文件能明确提示。

## Milestone 3：同步播放核心

目标：两端播放状态稳定同步。

交付：

- WebRTC DataChannel。
- 播放命令协议。
- 房间时间同步。
- 漂移修正。
- 主控模式。
- 自由模式。

验收：

- play/pause/seek/rate 能同步。
- 正常网络下长期漂移小于 250ms。
- 断线重连后能恢复当前播放进度。

## Milestone 4：追剧体验

目标：完整支持一起追剧。

交付：

- 房间播放列表。
- 剧集识别。
- 手动下一集。
- 自动下一集。
- 双方缺集检查。
- 同步表情反应。

验收：

- 切集前能确认双方都有文件。
- 自动下一集不会把一方切到不存在的媒体。
- 表情反应不影响播放同步。

## Milestone 5：内测发布

目标：交给真实用户小范围试用。

交付：

- Windows 安装包。
- 基础错误日志。
- 隐私说明。
- 连接诊断页。
- 内测反馈模板。

验收：

- 非开发环境可安装运行。
- 常见连接失败能给出诊断信息。
